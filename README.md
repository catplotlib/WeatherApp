# WeatherApp
A weather app made in react native that shows the current teperature and weather and forecasts temperature for the next two days.The wether data is fetched from https://www.weatherapi.com/ ,styling is done using styled components and the app is published using expo.

# Installation
Clone the repository and navigate to the folder. Install the dependencies and run the file.
```
npm i
npm start
```
